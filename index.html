개인정보처리방침

Recipesoup

1. 개인정보 처리 현황

Recipesoup은 사용자의 개인정보를 소중히 여기며 개인정보보호법에 따라 다음과 같이 개인정보 처리방침을 알려드립니다.

수집하는 개인정보

본 앱은 회원가입이나 계정 기반 서비스를 운영하지 않으며 사용자가 입력한 레시피·감정 기록·설정 정보는 기본적으로 기기 내부에만 저장됩니다.

다만, 사용자가 선택적으로 이용하는 AI 분석 기능 사용 시 일부 정보가 외부 서비스로 전송될 수 있습니다. 해당 항목은 아래에서 상세히 안내합니다.

데이터 저장 방식

  • 레시피·감정 기록·토끼굴 진행상황·앱 설정은 기기의 로컬 데이터베이스에 저장됩니다.
  • AI 분석용 사진과 텍스트는 분석 결과만 기기에 저장되며 원본은 유지되지 않습니다.

2. 외부 서비스 이용

OpenAI API 서비스

레시피 분석 및 추천 기능을 지원할 때 다음과 같은 데이터가 사용됩니다.

  • 전송 데이터: 사용자가 입력한 레시피 텍스트·URL·감정 메모·음식 사진 등 분석 요청에 포함한 내용입니다.
  • 처리 목적: 레시피 추출, 재료 분석, 조리법 정리 및 추천 결과를 제공하기 위함입니다.
  • 보관 방식: OpenAI는 요청을 처리하는 동안 데이터를 일시적으로 보유할 수 있으며 앱은 해당 원본 데이터를 저장하거나 별도로 보관하지 않습니다.
  • 개인식별정보: 텍스트와 이미지에 민감한 개인정보를 포함하지 않도록 입력을 피해 주시기 바랍니다.

중요: 앱은 OpenAI API 응답으로 생성된 결과만 로컬에 저장하며 사용자가 제출한 원본 텍스트·이미지는 별도로 보관하지 않습니다.

클라우드 콘텐츠 업데이트

홈 화면의 "제철 레시피/요리 상식/콘텐츠 큐레이션" 섹션은 운영자가 관리하는 클라우드 콘텐츠 저장소에서 최신 정보를 가져옵니다.

  • 전송 데이터: 기기에서 콘텐츠를 요청하는 데 필요한 최소한의 식별 값(예: 앱 버전, 언어 설정)이 전송될 수 있으며, 사용자 작성 데이터는 포함되지 않습니다.
  • 처리 목적: 최신 큐레이션 콘텐츠를 제공하기 위함입니다.
  • 보관 방식: 요청 시 서버에서 콘텐츠만 내려보내고 사용자 작성 데이터는 수집하지 않습니다.

3. 데이터 보안

  • 사용자가 작성한 데이터는 로컬에 저장되며 외부 전송은 상기 안내한 선택 기능에 한정됩니다.
  • 전송 구간은 TLS 등 표준 암호화 방식을 사용하며 앱은 OpenAI 응답 결과만 저장합니다.
  • 앱 삭제 시 로컬 데이터가 함께 삭제되며 자동 동기화 기능은 제공하지 않습니다.

4. 사용자 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 앱 내 설정에서 데이터 삭제
  • 앱 삭제를 통한 모든 데이터 완전 삭제
  • 특정 레시피 또는 데이터 개별 삭제

5. 아동의 개인정보 보호

본 앱은 만 14세 미만 아동의 개인정보를 고의로 수집하지 않으며 앱에 입력한 내용은 기기에 저장된 상태로 관리됩니다.

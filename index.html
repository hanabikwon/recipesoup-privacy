개인정보처리방침

Recipesoup

1. 개인정보 처리 현황

Recipesoup은 사용자의 개인정보를 소중히 여기며, 개인정보보호법에 따라 다음과 같이 개인정보 처리방침을 알려드립니다.

수집하는 개인정보

본 앱은 개인정보를 수집하지 않습니다. 모든 데이터는 사용자의 기기에만 저장됩니다.

데이터 저장 방식

  • 레시피 정보: 사용자 기기의 로컬 데이터베이스에만 저장
  • 사진 및 이미지: 사용자 기기에만 저장
  • 앱 설정 및 환경설정: 사용자 기기에만 저장
  • 버로우(게임) 진행상황: 사용자 기기에만 저장

2. 외부 서비스 이용

OpenAI API 서비스

레시피 분석 및 추천 기능을 위해 OpenAI API를 사용합니다:

  • 전송 데이터: 사용자가 입력한 레시피 텍스트 또는 음식 사진
  • 처리 목적: 레시피 정보 추출, 재료 분석, 조리법 정리
  • 보관 기간: OpenAI는 API 요청 데이터를 임시적으로만 처리하며 저장하지 않습니다
  • 개인식별정보: 전송되지 않습니다

중요: OpenAI API로 전송되는 데이터에는 개인식별정보가 포함되지 않으며, 순수하게 레시피 관련 정보만 전송됩니다.

3. 데이터 보안

  • 모든 사용자 데이터는 기기에서만 관리됩니다
  • 외부 서버로 개인정보가 전송되지 않습니다
  • 앱 삭제 시 모든 데이터가 함께 삭제됩니다
  • 기기 간 데이터 동기화는 지원하지 않습니다

4. 사용자 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 앱 내 설정에서 데이터 삭제
  • 앱 삭제를 통한 모든 데이터 완전 삭제
  • 특정 레시피 또는 데이터 개별 삭제

5. 아동의 개인정보 보호

본 앱은 만 14세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 모든 데이터는 기기에만 저장되므로 추가적인 보호조치가 필요하지 않습니다.

6. 개인정보처리방침 변경

본 방침이 변경될 경우, 앱 업데이트를 통해 공지하며 중요한 변경사항은 별도 안내할 예정입니다.

7. 문의사항

개인정보 처리방침에 관한 문의사항이 있으시면 다음 연락처로 문의해 주시기 바랍니다:

  • 개인정보보호 책임자: Recipesoup 개발팀
  • 연락방법: 앱 내 설정 > 앱 정보 > 문의하기
